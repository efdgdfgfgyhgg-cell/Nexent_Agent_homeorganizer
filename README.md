# HomeOrganizer 居家收纳师 🏠

> **全屋收纳整理规划多智能体系统——让每件东西都有合适的位置**

基于 Nexent 平台构建的纯多智能体协作系统，不依赖任何 MCP 工具，完全通过智能体间 prompt 协作完成全屋收纳方案输出。

## 架构总览

系统由 **1 个主智能体 + 5 个子智能体** 组成，按照"诊断 → 断舍离 → 设计 → 优化 → 采购"的标准流程协作。

```
┌─────────────────────────────────────────────────────────────────┐
│                    居家收纳师 (主智能体)                         │
│                home_organizer_master                           │
│        负责理解需求、调度子智能体、整合输出完整方案              │
└──┬──────────┬──────────┬──────────┬──────────┬─────────────────┘
   │          │          │          │          │
   ▼          ▼          ▼          ▼          ▼
┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
│空间   │  │收纳   │  │断舍离 │  │动线   │  │采购   │
│诊断师  │  │方案师  │  │教练   │  │优化师  │  │顾问   │
└──────┘  └──────┘  └──────┘  └──────┘  └──────┘
找问题    出方案    减物品    顺流程    给清单
```

---

## 子智能体说明

| 序号 | 智能体 | 角色 | 核心职责 |
|------|--------|------|---------|
| 1 | 空间诊断师 | 🩺 | 分析空间现状，找出储物不足、动线混乱、空间浪费等问题 |
| 2 | 收纳方案师 | 📐 | 针对诊断结果设计具体收纳系统（柜内分区、工具选型） |
| 3 | 断舍离教练 | 🧹 | 帮助用户做物品分类决策，克服"扔不掉"心理 |
| 4 | 动线优化师 | 🚶 | 分析日常行动路线，减少无效往返，让生活更顺手 |
| 5 | 采购顾问 | 🛒 | 推荐收纳工具类型和价格区间，给出选购注意事项 |

## 协作流程

```
用户输入问题
    │
    ▼
主智能体判断需求范围（全屋 / 单空间 / 单品类）
    │
    ├─ 全屋需求 → 依次调用全部 5 个子智能体
    ├─ 单空间  → 调用空间诊断师 + 收纳方案师 + 采购顾问
    └─ 单品类  → 调用对应的 1-2 个子智能体
    │
    ▼
各子智能体按顺序协作：
  空间诊断师 → 断舍离教练 → 收纳方案师 → 动线优化师 → 采购顾问
    │
    ▼
主智能体整合输出完整方案
    │
    ├─ 一、问题诊断
    ├─ 二、断舍离建议
    ├─ 三、收纳设计方案
    ├─ 四、动线优化
    ├─ 五、采购清单
    ├─ 六、实施步骤
    └─ 七、日常维护建议
```

## 覆盖的场景

- **厨房**：台面小家电整理、地柜/吊柜空间优化、水槽下空间利用
- **客厅**：茶几杂物管理、电视柜收纳、墙面展示收纳结合
- **卧室**：衣柜空间规划、衣物分类、梳妆台/床头柜整理
- **玄关**：出门中转站建立、鞋柜扩容、钥匙/包/外套归位
- **卫生间**：洗漱区收纳、镜柜利用、清洁用品管理
- **书房/儿童房**：书架分类、学习用品收纳、空间分区

## 使用方式

1. 在 Nexent 平台中导入 `homeorganizer_main_agent.json`
2. 配置模型（推荐 DeepSeek-V3.2 或同等能力的模型）
3. 输入你的收纳需求，例如：
   - _"我家厨房台面全堆满了，做饭很挤，怎么办"_
   - _"两室一厅住了三年，东西越堆越多，想全屋整理"_
   - _"衣柜塞满了但每天觉得没衣服穿"_
4. 主智能体会自动判断需要哪些子智能体参与并输出完整方案

## 设计思路

本系统的设计灵感来源于专业的整理收纳流程与家庭生活动线研究。核心设计原则：

- **好拿、好放、好维持**：收纳方案不以"看起来整齐"为目标，而以"日常使用顺手"为标准
- **先减后加**：先断舍离再买收纳工具，避免用收纳工具堆积更多杂物
- **动线优先**：最常用的东西放在最顺手的位置，减少每天不必要的走动
- **渐进实施**：建议按区域分阶段执行，降低执行门槛
